Daum DNA는 다음 내부의 소스 코드를 공개하여, 외부 개발자들과 공동 개발 및 상생 협력에 앞장서고 있습니다. 특히, 협력이 가능한 프로젝트를 선별하여 외부 공개를 진행 중입니다.

Daum 오픈 에디터

Daum Editor 는 Daum 에서 개발된 WYSIWYG기반의 웹 에디터인 Daum Editor를 기반으로 한 Semantic Web Information Editor입니다.

기존 웹 에디터는 입력 데이터를 의미적으로 구분하여 입력 받거나 메타 데이터를 입력 받는 기능이 부재하여 사용자들의 콘텐츠 생산 욕구를 충분히 받아들이지 못하고 있는데 반해 Daum Editor는 OpenAPI, Microformat, RDF 등을 지원하여 시맨틱 콘텐츠 생산을 돕고 웹 데이터의 상호 연결성을 높여주고자 합니다.

  • 다운로드 : https://github.com/daumcorp/DaumEditor
  • 예제사이트: http://uie.daum.net/openeditor/sample/latest/editor.html

Daum API Code

Daum API Code는 오픈 API를 사용하는 일반 개발자 및 제휴 업체를 위하여 다음에서 제공하고 있는 모든 샘플 코드를 모아 둔 코드 저장소 입니다. 다양한 언어별 코드를 제공하고 있으며, 각 API별 튜토리얼 및 샘플코드로 이용되고 있습니다.

  • 다운로드 : https://github.com/daumdna/apis

Dough

Dough (도우)는 웹표준기술팀에서 제작한 CSS 기반 UI Framework 입니다. CSS 전문가와 비전문가 누구나 몇가지 규칙을 숙지하기만 하면 자유롭게 웹표준 기반 마크업을 개발할 수 있도록 돕는 것이 목적이며, 이미지 기반이 아닌 CSS 기반의 스타일링을 가지므로 개발자 입장에서 보다 손쉬운 유지관리가 가능합니다.

  • 프로젝트 사이트 : http://dough.daum.net
  • 소스코드 저장소: https://github.com/daumcorp/Dough
